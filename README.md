# GunsmithBot

Discord bot to retrieve weapon rolls for Destiny 2

## Setup

```bash
cp .env.sample .env
```

Fill out the `.env` file with the appropriate values.

## Installation

Node.js v16 is recommended.

```bash
npm install
```

## Usage

```bash
npm start
```
